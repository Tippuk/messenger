## Messenger (лабораторная №2)

Простой учебный мессенджер с архитектурой клиент‑сервер на C# и Windows Forms.

- `ChatServer` — сервер, который принимает подключения клиентов и рассылает сообщения.
- `ChatClient` — клиент с графическим интерфейсом (история чата, поле ввода, настройки подключения, список пользователей).

### Как запустить

1. Открыть solution `Messenger.sln` в Visual Studio.
2. Собрать решение.
3. Запустить проект `ChatServer` (сервер) и оставить его работать.
4. Запустить один или несколько экземпляров проекта `ChatClient`.
5. В клиенте указать:
   - IP сервера (по умолчанию `127.0.0.1`),
   - порт (по умолчанию `9000`),
   - ник пользователя.
6. Нажать «Войти», затем вводить сообщения и нажимать «Отправить».

Сообщения отображаются в формате: `[Время] ИмяПользователя: Текст сообщения`.

История сообщений сохраняется в `chat-log.json` рядом с исполняемым файлом клиента и загружается при следующем запуске.

### UML‑диаграмма

Отдельно нужно сделать UML‑диаграмму классов в draw.io (или аналогичном инструменте) на основе следующих основных классов:

- `Message`
- `ChatServerInstance`
- `ClientConnection`
- `ChatClientConnection`
- форма клиента `Form1` (содержит ссылки на элементы UI и объект `ChatClientConnection`).

Готовую диаграмму стоит сохранить в формате PNG/JPG и добавить в репозиторий (например, в папку `docs`).
