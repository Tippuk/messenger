## Messenger (лабораторная №2)
### Как запустить

1. Открыть solution `Messenger.sln` в VS.
2. Собрать решение.
3. Запустить проект `ChatServer` и оставить его работать.
4. Запустить один или несколько экземпляров `ChatClient`.
5. В клиенте указать:
   - IP сервера (по умолчанию `127.0.0.1`),
   - порт (по умолчанию `9000`),
   - ник.
6. Нажать «Войти», затем вводить сообщения и нажимать «Отправить».
